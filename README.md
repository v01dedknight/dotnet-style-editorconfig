# C# EditorConfig (2-space style)

Конфиг для автоматического соблюдения стайлгайда в C# проектах через `.editorconfig`.

Работает в Visual Studio, Rider и других IDE с поддержкой EditorConfig.

## Что настраивается

- Отступы 2 пробела
- Скобки на той же строке
- lowerCamelCase для локальных переменных
- Запрет на использование `var`
- Ошибки подсвечиваются как `error`

## Как использовать

1. Скопируйте файл `.editorconfig` в корень вашего проекта (через Solution Explorer).
2. Убедитесь, что файл лежит на уровне `.sln`.
3. Когда код готов — нажмите

   Ctrl + K, Ctrl + D

IDE автоматически
- выставит отступы в 2 пробела
- приведёт скобки к нужному виду
- подсветит нарушения нейминга
- подсветит использование `var`

## Зачем это нужно

Это не решает вопросы архитектуры,  
но снимает ~80% боли с оформления кода.
